## Описание классов, их полей и методов

Класс `Cell` - хранение параметров отдельный клеток на карте:

- конструктор `__init__(self, id, cell_height_coord, cell_width_coord, is_alive=False, neighbors_ids)`
- приватные поля:
    - `id` - идентификатор клетки в формате "hYwX", где Y - номер ряда карты (начиная с 1), X - номер столбца (начиная с 1);
    - `height_coord` - - номер ряда карты (начиная с 1);
    - `height_coord` - номер столбца (начиная с 1);
    - `is_alive` - статус клетки "жив"/"мертв" (по умолчанию `False`);
    - `neighbors_ids` - множество id формата "hYwX" всех соседних клеток, расположенных в 1 шаге от данной на конкретной карте;
    - `neighbors` - множество объектов-клеток, расположенных по соседствую на конкретной карте.
- Методы:
    - `set_cell_status(self, is_alive)` - предоставляет доступ к приватному полю `is_alive` для перезаписи значения;
    - `add_neighbors(self, neighbors_set)` - предоставляет доступ к приватному полю `neighbors` для записи множества соседних объектов-клеток на конкретной карте (при создании данное поле создается со значением `None`).
- Тесты:
    - констрyктор;
    - метод `set_cell_status`;
    - метод `add_neighbors`.
